# Chatbot-NLP-de-classification-automatique-des-intentions-avec-NLTK-et-Streamlit
Chatbot intelligent basé sur le traitement du langage naturel (NLTK) capable de classifier automatiquement des messages en 38 intentions. Le modèle utilise TF-IDF et Naive Bayes. Une interface interactive est développée avec Streamlit et déployée via ngrok pour une utilisation en temps réel.
